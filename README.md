# Personal Portfolio

This is my personal portfolio website showcasing my projects, skills, and contact information. It's built with Next.js, TypeScript, and Tailwind CSS.

What I do in this project

- Build responsive, accessible web interfaces using Next.js and TypeScript.
- Design and implement UI/UX with Tailwind CSS and custom components.
- Create interactive components and small animations (typewriter effect, theme toggle, grain overlay).
- Organize and present project case studies with tech stacks, links, and images.
- Manage project data in `src/data/projects.ts` and media in the `public/` folder.
- Optimize for performance and SEO (image optimization, metadata, clean markup).
- Maintain the site with Git: feature branches, commits, and deployments.

Project structure (key folders)

- `src/app/` — Next.js app routes and pages (home, about, contact, projects).
- `src/components/` — Reusable UI components (Navbar, ThemeProvider, TypewriterEffect, GrainOverlay).
- `src/data/` — Project data and content sources.
- `public/` — Static assets and images.

Live demo

Check the deployed site on Netlify: https://vanshsuthar022.netlify.app/

Run locally

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```
